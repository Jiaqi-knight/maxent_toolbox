Introduction

Maximum entropy toolbox for MATLAB is a toolbox for fitting maximum entropy models to experimental data. The toolbox was originally built for learning the probability distribution of population spiking activity in neural recordings.

Documentation

If this is the first time you are using the toolbox, start by reading the quick start guide.

Existing users may want to refer to the function reference for details about each of the available functions in the toolbox.
